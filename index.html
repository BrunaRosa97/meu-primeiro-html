História da Web

O computador Harvard Mark 1 (eletromecânico), criado no início de 1944 para gerar cálculos, ficou marcado como o início da era na História da Web.
Logo em seguida estando nessa mesma geração veio Colossus (1946) e Eniac (automático-1946) ambos para cripto análises na época da guerra.
A 2ª geração (1959-1965) veio marcada pelos transitores, fazendo com que os computadores diminuíssem de tamanho.
A 3ª geração (1965-1970) foi marcada pelos circuitos integrados.
Já a 4ª geração (a partir 1970) se deu o início do uso de microprocessadores, época em que foi criado também os mouses e Pcs.
Há rumores que já estamos chegando na 5ª geração que será marcada pela Inteligência Artificial, criptomoedas, computadores quânticos, metaverso, etc...

Surgimento da Internet

Joseph Carl Robnett Licklider teorizou sore uma rede onde todos poderiam se comunicar através dela.
Algum tempo depois foi criado então o ARPANET, um Sistema de Comunicações em Pacotes, enviados um a um que permitia múltiplos destinos.
A primeira conexão estabelecida foi feita em 29/10/1969; e em 1989, Tim Berners-Lee viu a oportunidade de unir hipertexto com TCP/IP e criar a World Wide Web (www), sendo o primeiro navegador em 06/08/1991.

Clients e Servers

Em Clients o cliente é o computador que solicita os serviços e recursos de um servidor dentro de um sistema computacional.
Já o Server (servidor) é um recurso dentro de um sistema computacional, capaz de processar aplicações, armazenar dados e prestar serviços.
Dentro desse universo teremos:
Navegadores que são programas criados por empresas, utilizados para abrir/executar arquivos (também chamados de browsers);
Aplicação Web que são soluções criadas que possuem a internet como meio de comunicação entre client x server não sendo necessário sua instalação;
Hardwares(parte física) e Softwares(parte lógica) para ajudar os servidores no armazenamento, memória, processadores, sistemas operacionais, monitaramento e servidores web.

Linguagem de Programação

É uma linguagem escrita e formal que especifica um conjunto de instruções e regras usadas para gerar programas (software).
Linguagem Server-side(backend) são linguagens que o servidor entende. Isso quer dizer que vai escrever um código onde o servidor vai processá-lo e então vai mandar para o seu navegador a resposta; linguagens executadas no servidor: PHP, JAVA, RUBY, C#, JAVASCRIPT(node.js), PYTHON...
Linguagem Client-side(frontend) são linguagens onde apenas o seu navegador vai entender; Linguagem executada no navegador: JAVASCRIPT.
